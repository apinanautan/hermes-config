# Secretary Tasks — Structure

## Directory Layout
```
docs/secretary_tasks/
  inbox/        → งานรอทำ
  in_progress/  → กำลังทำ
  done/         → เสร็จแล้ว
  reports/      → reports
```

## Flow
```
inbox → in_progress → report → done
```

## Task Format
```markdown
# Task: <TASK_ID>
- target: Hermes-PC | Hermes-Mac
- status: pending | done
```
